# pokeplanner

## Problemstellung
For this project, we decided to develop an application that helps Pokémon players plan their teams. 
While existing tools like [Pokémon Team Planner](https://richi3f.github.io/pokemon-team-planner/) are helpful, they aren't optimized for mobile devices. 
Furthermore, they often lack a user-friendly way to create and evaluate teams, and the mobile view is missing useful information.

## Lösung
We are developing a Pokémon Team Planner where you start by searching for your desired game. Once selected, the app displays all Pokémon obtainable in that specific version.
By tapping on them, you can add Pokémon to your team. A dedicated button then allows you to analyze your team composition—for example, in terms of type coverage and weaknesses.
The app is being built with React.

## Requirements
- Node v23

## Projekt Setup
1. Clone repo
2. Navigate to "planner"
3. Open terminal and execute `npm install`
4. Start site with `npm run dev`
